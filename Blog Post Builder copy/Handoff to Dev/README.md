# Handoff to Dev — Daily Blog Drops

Each morning the blog task creates a dated folder here, e.g. `2026-06-16/`, containing that day's finished blogs ready to publish.

**Per blog, the folder contains:**
- `[slug].md` — the full blog: title, meta description, URL slug, body (1,200-1,800 words, H2/H3), internal links marked, closing CTA.
- A **HERO IMAGE block** at the top of each file with: the free-license image's direct URL, the source page, the license (e.g. CC BY 4.0 / Public Domain / Unsplash License), and the exact **attribution line** to display. The dev downloads, hosts, and credits it.
- **Tagging note** — which lake/town the post must be tagged to so it auto-populates that page's "From the blog" module.

**Daily index:** each dated folder also has an `INDEX.md` listing the day's posts, their target money pages, and tags — the copy-paste handoff for the dev.

Nothing here is published automatically; these are drafts for the dev to ship.
