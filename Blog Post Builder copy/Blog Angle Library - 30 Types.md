# Blog Angle Library — 30 Reusable Types
**For any lake or town. Swap `[Lake]` / `[Town]` and ship.**
_Owner: Director of Marketing · Templatable for Common Realtor (any geo)._

Each angle = a title pattern + who it's for + the page it boosts. Every blog gets **tagged to its lake/town** (fills the "From the blog" module) and links **up** to that money page 2-3x. Rotate angles so no two blogs on the same lake feel interchangeable.

Intent key: **INFO** = pulls long-tail search traffic · **BUY/SELL** = closer to a lead · **LOCAL** = local-authority + freshness signal.

### How to use
- **Lake page** → rotate from angles 1-26. Start each lake with a buyer guide (#8), a lifestyle piece (#1 or #7), and a comparison (#26).
- **Town page** → use 27-30, plus pull in the lakes near it via #29.
- **Recurring:** the market report (#23) is the one to re-run quarterly per lake — fresh content + a reason for buyers to come back.
